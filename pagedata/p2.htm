Defining the Problem

Write a program to play the children’s game BizzBuzz.

The problem has already been defined for you: Print out the numbers from 1 to 100 following the BizzBuzz rules.

You know the input (the numbers 1–100), the processing (check to see if the number is a multiple of 3 or 5 or both), and the output (bizz, buzz, bizzbuzz, or the number).

So let’s start designing the solution.
